Основи мови C#

C# — це сучасна об’єктно-орієнтована мова програмування, створена компанією Microsoft для платформи .NET.

Змінні та типи даних

У C# існують різні типи даних: int, double, string, bool та інші.

Методи

Метод — це блок коду, який виконує певну дію. Його можна викликати з інших частин програми.

Класи та об’єкти

Клас містить дані (поля) і поведінку (методи). Об’єкт — це екземпляр класу.

Приклад оголошення змінної

Наприклад: int age = 25;

Конструктори

Конструктор — це спеціальний метод, який викликається при створенні нового об’єкта.
